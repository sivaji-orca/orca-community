# Migrating from Orca Community to Orca Cloud

## When to Upgrade

Orca Community Edition is great for individual developers. Consider upgrading to Orca Cloud when you need:

- **Team Collaboration** — Multiple developers working on the same MuleSoft projects
- **Centralized Dashboard** — Cloud-hosted monitoring, deployments, and API management
- **Managed Integrations** — Anypoint Platform and Salesforce connections managed for you
- **Role-Based Access** — Owner, admin, developer, and viewer roles
- **Audit Trail** — Full history of who deployed what and when
- **Professional Support** — Priority support from the Orca team

## How to Migrate

1. **Sign up** at [orcaesb.com](https://orcaesb.com/sign-up)
2. **Create your organization** in the Orca Cloud dashboard
3. **Import projects** — Your existing MuleSoft project configurations transfer directly
4. **Connect Anypoint Platform** — Add your Connected App credentials in Settings
5. **Invite your team** — Add team members with appropriate roles

## What Changes

| Feature | Community | Cloud |
|---------|-----------|-------|
| Dashboard | Local (localhost:5173) | orcaesb.com/dashboard |
| Projects | Up to unlimited | Free: 3, Pro: 50+ |
| Team Members | Single user | Free: 5, Pro: 100+ |
| Deployments | Manual CLI/Dashboard | One-click from UI |
| Monitoring | Local logs | Real-time Anypoint metrics |
| Salesforce | Manual config | OAuth one-click connect |
| Support | GitHub Issues | Priority email + Slack |
