## MuleSoft Community Forum Post

**Title:** Introducing Orca Community Edition - Free Open-Source Developer Dashboard for MuleSoft Teams

**Body:**

Hi MuleSoft community!

I'm excited to share **Orca Community Edition** — a free, open-source developer productivity tool specifically built for MuleSoft teams.

### The Problem
MuleSoft development involves juggling multiple tools: Anypoint Studio, CloudHub, Postman, Git, Salesforce CLI, Maven, and more. Switching between them slows down development and makes onboarding new team members harder.

### The Solution
Orca Community Edition is a self-hosted dashboard that unifies your MuleSoft development workflow:

1. **API Design** — RAML spec management
2. **Project Explorer** — Browse and manage your Mule projects
3. **New Project Wizard** — Scaffold a complete Mule 4 project with one click
4. **Deploy** — One-click local and CloudHub deployment
5. **Monitoring** — Real-time health probes for all your APIs
6. **Logs** — Live Mule Runtime log streaming with filtering
7. **Analytics** — API metrics (requests, errors, response times)
8. **Postman** — Auto-generate collections and environments
9. **Git** — Commit, push, and create PRs from the dashboard
10. **Settings** — Configure Anypoint, Salesforce, GitHub, Postman credentials

### Getting Started
```bash
git clone https://github.com/sivaji-orca/orca-community.git
cd orca-community
./scripts/setup.sh
./scripts/start.sh
```

### Tech Stack
- Backend: Bun + Express + SQLite
- Frontend: React 19 + Tailwind CSS v4
- Runtime: Mule 4.11.2
- AI: Cursor IDE rules for MuleSoft patterns

### Links
- **GitHub**: https://github.com/sivaji-orca/orca-community
- **Website**: https://orcaesb.com/community
- **Docs**: See the repo README and /docs folder

We'd love your feedback! Star the repo, open issues, or contribute.
